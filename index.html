    • Select the size of the Grid by changing the values in the option fields.

      • Expect the canvas to change size depending on the values you input.


    • Then Pick the color you'd want to paint with. Click on the color picker to select the color.

    • Left click your mouse and press, then drag to paint over the canvas. The color you selected will be painted on the canvas.

      • You can erase the color in a cell by right clicking on your mouse. Use the reset button to clear input values.
